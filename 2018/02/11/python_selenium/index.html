Python+Selenium进行动态加载网页爬取

2018-02-11

一、前言：

今天想要爬取PTA的题目信息，但是发现scrapy爬取返回的response中没有题目信息，检查元素查看
了一下PTA题目网页的源代码，发现这个网页是使用js动态加载的，也就是说是需要浏览器加载好js
代码，返回题目信息。那怎么进行爬虫呢，这就需要使用前几章提到的Selenium，本文使用python+
Selenium进行动态加载网页爬取的演示和讲解。

二、Selenium简介：

selenium 是一套完整的web应用程序测试系统，包含了测试的录制（selenium IDE）,编写及运行
（Selenium Remote Control）和测试的并行处理（Selenium Grid）。Selenium的核心Selenium Core基
于JsUnit，完全由JavaScript编写，因此可以用于任何支持JavaScript的浏览器上。

selenium可以模拟真实浏览器，自动化测试工具，支持多种浏览器，爬虫中主要用来解决
JavaScript渲染问题。

三、Selenium进行爬取：

  1. 声明浏览器对象（你使用的浏览器）
browser = webdriver.Chrome()

Selenium可以声明很多种浏览器对象，可以模拟各种浏览器，需要对应浏览器类型及其版本下载
相应的webdriver，并将webdriver的目录加入到系统的环境变量中。

Selenium 支持的伪浏览器： Chrome, Firefox, IE, Opera, HtmlUnit, PhantomJS, Appium等

  1. 访问抓取页面
browser.get("http://www.baidu.com")
  1. 查找元素

查找元素的方式有很多种，可以进行单个元素或多个元素的查询，多个元素查询返回查询列表

（1） 单个元素查询的不同方式：

find_element_by_name
find_element_by_id
find_element_by_xpath
find_element_by_link_text
find_element_by_partial_link_text
find_element_by_tag_name
find_element_by_class_name
find_element_by_css_selector

（2） 多个元素查询的不同方式：

find_elements_by_name
find_elements_by_id
find_elements_by_xpath
find_elements_by_link_text
find_elements_by_partial_link_text
find_elements_by_tag_name
find_elements_by_class_name
find_elements_by_css_selector

发现了什么，多个元素查询只是将单个元素中的”element”修改为了”elements”，是不是很简单。

xpath语法什么的大家都使用过，网上资料也很多，这里贴个爬取PTA的例子，作为参考。

#-*- coding: utf-8 -*-
from selenium import webdriver
import codecs

chrome_options = webdriver.ChromeOptions()
chrome_options.add_argument('--headless')
chrome_options.add_argument('--disable-gpu')

browser = webdriver.Chrome(chrome_options=chrome_options)

browser.get("https://pintia.cn/problem-sets/14/problems/735")

title = browser.find_element_by_xpath('//div[@class="row"]/div/div/h4')
ques = browser.find_element_by_xpath('//div[@class="ques-view"]/p[1]')
func_title = browser.find_element_by_xpath('//div[@class="ques-view"]/h3[1]')
func_code = browser.find_element_by_xpath('//code[@class="lang-c++"][1]')
func_text = browser.find_element_by_xpath('//div[@class="ques-view"]/p[2]')
test_title = browser.find_element_by_xpath('//div[@class="ques-view"]/h3[2]')
test_code = browser.find_element_by_xpath('//div[@class="ques-view"]/pre[2]/code[@class="lang-c++"]')
input_title = browser.find_element_by_xpath('//div[@class="ques-view"]/h3[3]')
input_code = browser.find_element_by_xpath('//code[@class="lang-in"]')
output_title = browser.find_element_by_xpath('//div[@class="ques-view"]/h3[4]')
output_code = browser.find_element_by_xpath('//code[@class="lang-out"]')

res1 = title.text+'\n'+ques.text
res2 = func_title.text+'\n'+func_code.text+'\n'+func_text.text
res3 = test_title.text+'\n'+test_code.text
res4 = input_title.text+'\n'+input_code.text+'\n'+output_title.text+'\n'+output_code.text

res = res1+'\n'+res2+'\n'+res3+'\n'+res4

f=codecs.open(r'C:\Users\lovebear96\Desktop\1.txt','w','utf-8')
f.write(res)
f.close()

browser.close()
  1. 多说一句

代码中出现了–headless，这个是chrome的无浏览器模式，可以不用打开浏览器进行爬取，可以节
省资源和加快爬取速度。这个功能只有在高版本的chrome中才支持:

linux： Chrome 59+，Windows： Chrome 60+

四、总结

Selenium的功能很强大，他是模拟浏览器对网站进行解析，除了上面介绍的以外，还可以对网站进
行自动填充信息，自动提交等功能，有兴趣的同学可以查阅Selenium文档：

http://selenium-python.readthedocs.io/

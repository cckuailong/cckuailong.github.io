Ubuntu 14中忘记root password的解决办法（半转载）

2017-09-03

一、前言：

一个学长不小心把自己Ubuntu的root密码忘记了，这也唤起了我的好奇心，于是上网查了
一波，在此做一个记录，以后遇到这类尴尬（逗比）的情况，就能及时弥补了。

二、方法一：

如果用户具有sudo权限，那么直接可以运行如下命令：

sudo su root

passwd

更改密码

或者直接运行

sudo passwd root

命令就可以直接更改root密码。

有关sudo su的区别：

1.共同点：都是root用户的权限；

2.不同点：su仅仅取得root权限，工作环境不变，还是在切换之前用户的工作环境；sudo是
完全取得root的权限和root的工作环境。

三、方法二（大部分情况得使用此方法）：

如果用户不具备sudo权限，则方法一不能用，并需进入GRUB修改kernel镜像启动参数。本文
使用的Ubuntu版本为14.04.4，具体过程如下为：

1、重启电脑长按shift键直到进入下图进入GRUB引导模式，选择第二行Ubuntu 高级选项, 选中
直接回车

2、进入如下画面，看到里面有一些选项，这时千万不要按回车键，按e进入（recovery mode）
编译kernel进行启动参数

3、关键的时候到了， 倒数第四行，会看到一行

Linux /boot/vmlnuz-4.2.0-27-generic .......ro recovery nomodeset，

删除recovery nomodeset

4、然后再刚才删除的后面添加 quiet splash rw init=/bin/bash。然后按F10, 启动。

5、如果一切争取，运行后系统直接进入root mode，输入：passwd，系统会提示你输入新的密码，结束。

四、总结：

修改也比较的简单，但是尽量还是记住自己的root密码吧。

五、原文链接：

http://www.linuxidc.com/Linux/2016-05/131256.htm

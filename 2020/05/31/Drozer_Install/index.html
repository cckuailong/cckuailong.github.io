Windows 安装安卓渗透测试框架--Drozer

2020-05-31

一、前言

安装安卓渗透测试框架–Drozer基于python2，在windows环境下安装时会出现许多问题，
本文介绍Drozer的安装及简单使用方法，并配合夜神模拟器进行连接。

二、Python2.7 安装

由于 Python2 官方已经停止更新，已弃用一段时间，但是无奈 Drozer 运行基于 Python2，
所以先介绍，在已有Python3的环境下，安装 Python2 的方式（有坑）

  • 安装最后更新的 Python 2.7.18 版本 官网下载
  1. 下载msi文件进行安装，本文安装目录选择为 D:\Python27
  • 配置 Python2 使之与 Python3 共存
  1. 进入安装目录 D:\Python27，修改python.exe, pythonw.exe 为 python2.exe, pythonw2.exe
  2. 将 D:\Python27 和 D:\Python27\Scripts 加入到环境变量中，重启cmd生效

这时，打开cmd，输入python2，即为 Python 2.7.18 的运行环境

注：这里一定要注意，安装 python3 时系统变量中可能会有PYTHONHOME，指向 Python3 安装目录，要将这个系统变量删掉

  • 安装 Python2 对应的 easy_install 和 pip
  1. 下载 get-pip.py 文件 下载地址
  2. 使用 python2 安装
python2 get-pip.py
  1. 进入 D:\Python27\Scripts，删除pip.exe （重要！！为了与Python3-pip区分开）

这时，打开cmd，输入pip2，即为 Pip2的运行文件

安装 Drozer

  • 下载 msi安装包 官网下载
  • 安装时，选择安装到 Python2 的安装目录中（重要！！）
  • 安装必要组件
pip2 install protobuf
pip2 install pyOpenSSL
pip2 install twisted
pip2 install service_identity

安装完毕后，D:\Python27\Script中会多出4个 Drozer 相关的文件

安装夜神模拟器

自行安装，本文安装目录为 D:\Program Files\Nox

Drozer 连接 夜神模拟器

  • 模拟器中安装 agent
  1. D:\Python27\Lib\site-packages\drozer\lib 中的 standard-agent.apk 拖到夜神模拟器中进行安装
  2. 打开安装的 Agent，上方打开 Enable 按钮
  • adb 连接设备并转发流量到 Drozer Agent
  1. 使用夜神模拟器自带的 adb（重要！！其他可能不兼容，有冲突），在 D:\Program Files\Nox\bin 中，为了方便，把这个目录加入到环境变量
  2. adb 连接夜神模拟器
adb connect 127.0.0.1:62001
  1. 转发流量到 Drozer
adb forward tcp:31415 tcp:31415
  1. 进入 Drozer 安装目录， 本文为 D:\Python27\Script，运行
drozer.bat console connect

出现安卓图标即为安装成功

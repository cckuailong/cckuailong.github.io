Hexo的Yilia主题配置Valine评论功能

2019-12-15

一、前言

Hexo Your Blog教我们搭建自己的Hexo博客，使用的主题是Yilia。
由于多说等Yilia自兼容的评论系统已经跑路，所以选择为其配置Valine来满足需求。

二、配置LeanCloud

Valine是配合LeanCloud使用的，所以首先

  • 注册LeanCloud账号并登录控制台

  • 点击 “创建应用” – 输入应用名字（随便起） – 选择 “开发版” – “创建”

  • 点击创建好的应用的右上角的 “设置”(小齿轮)

“应用 keys” 中，有接下来会用到的 AppID 和 AppKey

进入 “安全中心”，”服务开关” 只开启数据存储，Web安全域名，添加Blog网站的域名

三、配置Yilia Valine

  • 新建 valine.ejs

进入hexo目录下的themes/yilia/layout/_partial/post，新建文件valine.ejs,写入

<div id="vcomment" class="comment"></div>
<script src="//cdn1.lncld.net/static/js/3.0.4/av-min.js"></script>
<script src='//unpkg.com/valine/dist/Valine.min.js'></script>
<script src="https://cdnjs.loli.net/ajax/libs/jquery/3.2.1/jquery.min.js"></script>
<script>
    var notify = '<%= theme.valine.notify %>' == true ? true : false;
    var verify = '<%= theme.valine.verify %>' == true ? true : false;
    new Valine({
        av: AV,
        el: '#vcomment',
        notify: notify,
        app_id: "<%= theme.valine.appid %>",
        app_key: "<%= theme.valine.appkey %>",
        placeholder: "<%= theme.valine.placeholder %>",
        avatar:"<%= theme.valine.avatar %>",
    });
</script>
  • 修改 article.ejs

在任意处加上

<% if (!index){ %>
  <% if (theme.valine && theme.valine.appid && theme.valine.appkey){ %>
    <section id="comments" class="comments">
      <style>
        .comments{margin:30px;padding:10px;background:rgb(255, 255, 255)}
        @media screen and (max-width:400px){.comments{margin:auto;padding:10px;background:rgb(214, 252, 203)}}
      </style>
      <%- partial('post/valine', {
        title: post.title,
        url: config.url+url_for(post.path)
      }) %>
    </section>
  <% } %>
<% } %>
  • 修改 Yilia配置文件(themes/yilia/_config.yml)

添加

# Valine https://valine.js.org
valine: 
  appid: LeanCloud中的AppID
  appkey: LeanCloud中的AppKey
  verify: false #验证码
  notify: false #评论回复提醒
  avatar: retro #评论列表头像样式：''/mm/identicon/monsterid/wavatar/retro/hide
  placeholder: 请留下宝贵的评论 #评论框占位符
  • 提交即可

hexo d

登陆LeanCloud控制台，存储–结构化数据–Comment，即可对评论进行操作

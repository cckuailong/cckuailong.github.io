Spark-HadoopYarn-Kafka集群搭建--基础环境

2019-12-08

前言

首先，是基础环境的搭建，包括，root账户设置，修改默认python版本为py3，配置域名，
配置ssh免密登陆，安装jdk8等。

配置 ubuntu 18.04.3

设置root账户 (master, slave*)

  • 激活root账户
sudo passwd -u root
sudo passwd root

  • 输入root密码

  • 现在可以root登陆了，但是要ssh开通root账号登陆

sudo vim /etc/ssh/sshd_config
  • 修改2处配置为 yes（记得取消注释）
PermitRootLogin yes
StrictModes yes
  • 重启ssh服务
service sshd restart

修改 apt 为国内软件源

  • 更换国内源

ubuntu 更换apt国内源 （阿里云镜像）

  • 更新软件
apt upgrade

修改python默认为python3

update-alternatives --install /usr/bin/python python /usr/bin/python3.6 1

配置域名 (master, slave*)

  • 修改hosts文件
vim /etc/hosts
  • 增加 master, slave* 的域名，例
192.168.139.138 master
192.168.139.139 slave1
192.168.139.140 slave2
... ...

ssh免密登陆

  • master，各slave，生成ssh rsa公私钥
ssh-keygen -t rsa

一路回车
  • slave* 生成的id_rsa.pub 发送给master
scp ~/.ssh/id_rsa.pub spark@master:~/.ssh/id_rsa.pub.slave1
  • master上将所有公钥加到authorized_keys中
cat ~/.ssh/id_rsa.pub* >> ~/.ssh/authorized_keys
  • master上生成的authorized_keys分发给每个slave
scp ~/.ssh/authorized_keys spark@slave1:~/.ssh/

安装jdk 8

  1. 安装openjdk-8
apt -y install openjdk-8-jdk
  1. 切换jdk版本
update-alternatives --config java

选择openjdk-8

  1. 设置环境变量

/etc/profile

export JAVA_HOME=/usr/lib/jvm/java-8-openjdk-amd64
export JRE_HOME=$JAVA_HOME/jre
export CLASSPATH=$JAVA_HOME/lib:$JRE_HOME/lib:$CLASSPATH
export PATH=$JAVA_HOME/bin:$JRE_HOME/bin:$PATH

source /etc/profile
